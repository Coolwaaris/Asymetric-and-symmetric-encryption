# Asymetric-and-symmetric-encryption
Scripts for file encryption in 2 devices.
First use the new.sh file to create pairs of asymmetric keys and create plain_data.txt file and shares the file to the
-second system files like public.key, encrypted file and the signature made using the private key and encrypted file to confirm 
confirm the sender. 
You just have to change the IP in script for your system.
